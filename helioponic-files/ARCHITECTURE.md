## Technology Stack

### Frontend (helioponic-mobile)
- **Framework:** React Native with Expo SDK 57
- **Navigation:** @react-navigation (bottom tabs + native stack)
- **State:** Zustand + React Context
- **Charts:** react-native-gifted-charts
- **HTTP:** Fetch API with custom client
- **WebSocket:** Native WebSocket with reconnection

### Backend (helioponic-backend)
- **Framework:** FastAPI (Python 3.12)
- **Database:** MongoDB 7 with Motor async driver
- **MQTT:** gmqtt (async)
- **Auth:** JWT (python-jose) + bcrypt (passlib)
- **Server:** Uvicorn (dev) / Gunicorn (prod)

### Firmware (helioponic-firmware)
- **ESP32:** Arduino framework, PubSubClient, ArduinoJson
- **Arduino Uno:** Bare-metal C++ (no external libraries)
- **Protocol:** UART Serial @ 9600 baud
- **Automation:** Bang-bang hysteresis (local edge control)

### Infrastructure
- **Containerization:** Docker Compose
- **MQTT Broker:** Eclipse Mosquitto 2
- **Networking:** Bridge network (helioponic-net)
